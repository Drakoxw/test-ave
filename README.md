# APP PRUEBA AVE

Abrir la App desplegada [ave-test-front](https://ave-test-front.netlify.app). 

## Iniciar
#### Instalar dependencias
```bash
   npm install
```
#### Correr app en local
```bash
   npm run dev
```

## Despliegue Netlify

#### Manual
   * Crear estaticos:
   ```bash
   npm run build
   ```
   * Ir a app.netlify.com/teams/< user >/sites
      > Arrastrar la carpeta ./dist dentro del `Drag and drop`

   * Abrir la Url
